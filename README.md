# E-commerce Sales Analysis Dashboard (Power BI)

## Project Overview
This project focuses on analyzing e-commerce sales data to identify key business insights such as overall performance, best-selling products, sales trends over time, and revenue contribution by category.
An interactive Power BI dashboard was developed to help stakeholders make data-driven decisions efficiently.

## Objectives
- Analyze overall sales, profit, and quantity performance

- Identify top-performing products based on total sales

- Understand sales trends by year and quarter

- Analyze revenue contribution by product category

- Enable interactivity using slicers for dynamic analysis

## Tools & Technologies
- Power BI Desktop

- Microsoft Excel / CSV

- DAX (Data Analysis Expressions)

- Power Query for data cleaning

## Dataset
The dataset contains e-commerce transactional data with the following key fields:

- Order Date

- Product Name

- Category & Sub-Category

- Sales, Profit, Quantity, Discount

- Region, State, City

## Key Insights
- Technology category contributes the highest share of revenue

- Sales show a growth trend over the years, peaking in later periods

- A small number of products drive a large portion of total sales

- Regional and category-level filtering reveals different performance patterns
